# DDUEksamen
Dette repository er dedikeret til et Eksamensprojekt i Teknikfag A (Digital Design og Udvikling)
Produktet er en browserudvidelse, hvor tanken bag var at oplyse brugere om deres privatliv på internettet.
Udvidelsen er sendt til Googles Webshop, i tanke om reel publicering.
Videreudviklingen af udvidelsen vil med stor sandsynlighed mindskes hvis ikke ophøre efter 1. Maj 2026.
